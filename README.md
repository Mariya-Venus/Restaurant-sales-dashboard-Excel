# 🍽️ Restaurant Business Performance Dashboard

## 📌 Project Overview

The Restaurant Business Performance Dashboard is an interactive Microsoft Excel dashboard developed to analyze restaurant sales, customer behavior, branch performance, and operational trends.

The dashboard transforms raw restaurant transaction data into meaningful business insights through KPI cards, charts, and interactive slicers.

This project helps management monitor sales performance, understand customer preferences, and make data-driven business decisions.

---

# 🎯 Business Objectives

The main objectives of this project are:

- Analyze restaurant sales performance
- Compare branch-wise revenue
- Identify top-selling menu items
- Study customer demographics
- Analyze payment methods and order channels
- Create an interactive and visually appealing dashboard
- Generate actionable business insights

# 📈 Key Business Insights

- Jumeirah generated the highest revenue.
- Fattoush Salad was the top-selling item.
- Dine-In contributed the highest share of orders.
- Customers aged 25–34 formed the largest customer segment.
- Card payments were the most preferred payment method.
